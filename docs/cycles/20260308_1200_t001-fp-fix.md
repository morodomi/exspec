- **A (pattern as "oracle shape") + B (.exspec.toml custom patterns)** is the right approach.
- T001 = "oracle-free" detection, not "assertion-free". Detect test oracles.
- Oracle shapes: root -> modifier chain -> terminal (call or property)
- Future: OracleKind enum as intermediate IR
